## Ответы на вопросы для самоконтроля

1. **Основные различия между development и production:**
   Development среда предназначена для разработки с подробными логами и отладкой, а production - для конечных пользователей с оптимизацией и безопасностью.

2. **Важность environment variables в production:**
   Environment variables обеспечивают безопасное хранение конфиденциальных данных и гибкость конфигурации без изменения кода.

3. **Организация управления разными средами:**
   Управление средами организуется через отдельные environment variables для dev/staging/prod и feature flags.

4. **Методы оптимизации производительности при деплое:**
   Оптимизация включает минификацию кода, сжатие статических файлов, кэширование и использование CDN.

5. **Настройка мониторинга ошибок в production:**
   Мониторинг настраивается через специализированные сервисы типа Sentry и системное логирование.

6. **Преимущества CI/CD:**
   CI/CD обеспечивает автоматизацию тестирования и деплоя для быстрых и надежных обновлений.

7. **Организация обновления без downtime:**
   Бесперебойные обновления организуются через blue-green деплой или canary релизы.

8. **Меры безопасности при деплое:**
   Ключевые меры безопасности включают HTTPS, защиту environment variables и правильные CORS настройки.
